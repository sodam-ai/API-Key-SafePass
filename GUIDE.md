## 이 문서를 읽기 전에

### 이 프로그램은 무엇을 하는 건가요?

혹시 이런 경험이 있으신가요? ChatGPT나 다른 AI 서비스, 또는 어떤 인터넷 서비스에 가입하면 "API 키"라는 길고 복잡한 암호 같은 글자(예: `sk-abc123...`처럼 생긴 것)를 받게 됩니다. 이걸 어디에 적어둬야 할지 몰라서 메모장 파일에 저장하거나, 카카오톡으로 자기 자신에게 보내두거나, 그냥 인터넷 창을 안 닫고 계속 켜두는 경우가 많습니다.

**API Key SafePass**는 이런 글자들을 **잠금장치가 달린 디지털 서랍**처럼 안전하게 모아 보관해주는 프로그램입니다. 딱 하나의 비밀번호(이걸 "마스터 비밀번호"라고 부릅니다)만 기억하면, 그 안에 저장해둔 모든 키를 안전하게 꺼내 쓸 수 있습니다.

### 컴퓨터를 잘 몰라도 따라 할 수 있나요?

네. 이 문서는 아래와 같은 것도 처음부터 설명합니다.

- **더블클릭**: 마우스 왼쪽 버튼을 아주 빠르게 두 번 연속으로 누르는 것을 말합니다. 프로그램이나 파일을 "열 때" 사용합니다.
- **폴더**: 컴퓨터 안에서 파일들을 정리해서 담아두는 노란색(또는 파란색) 상자 모양 아이콘입니다. 서랍이나 파일철과 비슷한 개념입니다.
- **다운로드**: 인터넷에서 내 컴퓨터로 파일을 가져오는 것을 말합니다. 보통 "다운로드" 폴더에 저장됩니다.
- **주소창**: 인터넷 브라우저(크롬, 엣지 등) 맨 위에 있는, 웹사이트 주소를 입력하는 긴 입력칸입니다.

### 이 프로그램은 안전한가요?

네. 이 프로그램은 **인터넷에 연결된 서버가 전혀 없습니다.** 내가 저장한 정보는 오직 내 컴퓨터 안에만 있고, 어디로도 전송되지 않습니다. 자세한 이유는 [README.md의 보안 섹션](./README.md#보안--데이터-흐름)에 기술적으로 설명되어 있지만, 이 가이드에서는 "사용법" 위주로만 설명합니다.

## 준비물

| 준비물 | 설명 |
|---|---|
| Windows 컴퓨터 | Windows 10 또는 Windows 11이 설치된 데스크톱/노트북. (스마트폰용은 아직 없습니다) |
| 인터넷 연결 | **1단계(다운로드)에서만** 필요합니다. 설치 후에는 인터넷을 꺼도 사용할 수 있습니다. |
| 잠깐의 시간 | 다운로드부터 첫 사용까지 5~10분 정도면 충분합니다. |

## 1단계 — 프로그램 내려받기

1. 컴퓨터에서 인터넷 브라우저를 엽니다. 보통 바탕화면이나 작업 표시줄(화면 맨 아래)에 있는 파란색 원 모양(Microsoft Edge) 또는 빨강·노랑·초록·파랑이 섞인 동그라미(Google Chrome) 아이콘을 더블클릭하면 됩니다.
2. 브라우저 창 맨 위, 글자를 입력할 수 있는 긴 칸(주소창)을 마우스로 한 번 클릭합니다.
3. 다음 주소를 정확히 입력하고 키보드의 `Enter` 키를 누릅니다.

   ```
   https://github.com/sodam-ai/API-Key-SafePass/releases
   ```

4. 화면에 "v0.1.0" 이라고 적힌 글자와 함께 초록색 "Latest" 표시가 보이는 페이지가 나타납니다.
5. 그 아래로 마우스 휠을 굴려 조금 내려가면 **"Assets"**(자산)이라고 적힌 부분이 보입니다. 그 글자를 클릭하면 파일 목록이 펼쳐집니다.
6. 목록에서 다음 파일을 찾아 클릭합니다. **파일 이름이 정확히 이래야 합니다.**

   ```
   API.Key.SafePass_0.1.0_x64-setup.exe
   ```

   > 왜 이 파일을 골라야 하나요? 이 파일이 "설치형"이라서, 프로그램 목록에 깔끔하게 등록되고 나중에 삭제하기도 쉽습니다. (다른 파일들의 차이는 [README.md의 다운로드 방법](./README.md#다운로드-방법)에서 확인할 수 있습니다.)

7. 클릭하면 자동으로 다운로드가 시작됩니다. 브라우저 화면 아래쪽이나 오른쪽 위에 "다운로드 중"이라는 표시가 뜹니다. 다운로드가 끝날 때까지 잠시 기다립니다(보통 몇 초면 끝납니다, 파일 크기는 약 1.8MB로 작습니다).
8. 다운로드가 끝나면 대부분 자동으로 "다운로드" 폴더에 파일이 저장됩니다. 잘 모르겠다면 키보드에서 `Windows 로고 키`를 누르고 "다운로드"라고 입력한 뒤 `Enter`를 누르면 다운로드 폴더가 열립니다.

## 2단계 — 설치하기

1. 방금 내려받은 `API.Key.SafePass_0.1.0_x64-setup.exe` 파일을 더블클릭합니다.
2. **여기서 파란색 화면이 뜨면서 "Windows에서 PC를 보호했습니다"라는 경고가 나타날 수 있습니다.** 당황하지 마세요. 이건 바이러스가 아니라, 이 프로그램이 아직 Windows에 "이 개발자는 안전합니다"라고 정식으로 등록하는 유료 절차(코드 서명 인증서)를 거치지 않았기 때문에 뜨는 표준 경고입니다.
   - 이 화면에서 **"추가 정보"**라는 회색 글자를 클릭합니다.
   - 그러면 아래에 **"실행"**이라는 버튼이 새로 나타납니다. 이 버튼을 클릭합니다.

   > 이 프로그램이 안전하다는 근거: 이 프로그램의 모든 내부 코드는 누구나 볼 수 있게 인터넷에 공개되어 있습니다(오픈소스). 전문가가 아니어도, "이 코드가 내 정보를 몰래 어딘가로 보내지 않는다"는 사실을 개발자가 [README.md의 보안 섹션](./README.md#보안--데이터-흐름)에서 구체적인 설정 파일 위치까지 밝혀두었습니다.

3. 설치 진행 화면이 나타납니다. 설치 위치를 고르는 화면이 나오면 특별히 바꿀 필요 없이 그대로 "다음" 또는 "설치" 버튼을 계속 눌러주면 됩니다.
4. "설치 완료" 또는 "마침" 버튼이 나오면 클릭합니다.
5. 이제 키보드의 `Windows 로고 키`를 누르고 "API Key SafePass"라고 입력하면 프로그램이 검색되어 나타납니다.

## 3단계 — 처음 실행하고 비밀번호 만들기

1. `Windows 로고 키`를 누르고 "API Key SafePass"를 입력한 뒤, 검색 결과에 나온 프로그램을 클릭(또는 더블클릭)해서 실행합니다.
2. 검은 배경에 하얀 글씨로 된 창이 열리고, 잠시 후 "마스터 비밀번호 설정" 같은 화면이 나타납니다.
3. **마스터 비밀번호란?** 이 프로그램 전체를 잠그고 여는 단 하나의 열쇠입니다. 은행 앱 비밀번호처럼, 다른 곳에서 쓰지 않는 나만의 비밀번호를 새로 만드는 것을 권장합니다.
4. 비밀번호를 입력하는 칸에 원하는 비밀번호를 입력합니다.
   - 너무 쉬운 비밀번호(`123456`, `password` 등)는 피하세요.
   - 최소 8자 이상, 숫자와 영문을 섞는 것을 권장합니다.
   - **이 비밀번호는 반드시 기억해야 합니다.** 어딘가 안전한 곳(머릿속, 또는 이 프로그램과 관계없는 다른 안전한 메모)에 남겨두는 것을 권장합니다.
5. 비밀번호를 확인하는 칸이 있다면 같은 비밀번호를 한 번 더 입력합니다.
6. "설정" 또는 "만들기" 버튼을 클릭합니다.

### 가장 중요한 순간 — 복구키

7. 비밀번호를 설정하자마자, 화면에 **"복구키"**라는 이름으로 영문/숫자가 섞인 긴 문자열이 딱 한 번 나타납니다.

   > **여기서 절대 그냥 "확인"을 누르고 넘어가지 마세요.**
   > 이 복구키는 마스터 비밀번호를 잊어버렸을 때 유일하게 다시 들어갈 수 있는 비상열쇠입니다. **이 화면은 나중에 다시 볼 수 없습니다.**

8. 아래 방법 중 하나로 반드시 복구키를 따로 저장하세요.
   - 종이에 손으로 옮겨 적어서 안전한 곳(수첩, 서랍 등)에 보관하기
   - 스마트폰으로 화면을 사진 찍어서, 이 프로그램과 무관한 안전한 곳(예: 다른 암호로 보호된 메모 앱)에 저장하기
   - `Windows 로고 키 + Shift + S`를 눌러 화면 일부를 캡처한 뒤, 캡처된 이미지를 안전한 폴더에 저장하기
9. 복구키를 안전하게 저장했다면, 화면의 "확인" 또는 "다음" 버튼을 클릭합니다.

## 4단계 — 첫 번째 키 저장해보기

1. 프로그램 화면 왼쪽에 있는 **"+"** 모양 버튼을 클릭해 프로젝트(정리함)를 하나 만듭니다. 이름은 자유롭게 정하면 됩니다(예: "내 첫 프로젝트").
2. 프로젝트를 만들고 나면, 화면 오른쪽 아래에 동그란 **"+"** 버튼이 보입니다. 이 버튼을 클릭합니다.
3. 키 등록 화면이 나타납니다. 아래 항목을 채워봅니다.
   - **이름**: 이 키가 무엇인지 알아볼 수 있는 이름 (예: "OpenAI 테스트 키")
   - **값**: 실제 API 키 문자열을 붙여넣거나 입력합니다.
   - 나머지 항목(제공자, 메모 등)은 선택 사항이니 비워둬도 됩니다.
4. "저장" 버튼을 클릭합니다. 목록에 방금 등록한 키가 나타나면 성공입니다.

## 5단계 — 저장한 키 찾아서 복사하기

1. 키보드에서 `Ctrl` 키를 누른 상태로 `K` 키를 함께 누릅니다(`Ctrl+K`).
2. 화면 위쪽에 검색창이 나타납니다. 키 이름의 일부를 입력하면 바로 아래에 결과가 나타납니다.
3. 원하는 키를 찾았다면, 키 옆에 있는 복사 아이콘(네모 두 개가 겹친 모양)을 클릭합니다.
4. 이제 이 값이 클립보드에 복사되었습니다. 사용하려는 곳(예: 다른 프로그램의 입력칸)에서 `Ctrl+V`를 눌러 붙여넣으면 됩니다.
5. 참고로, 복사한 값은 일정 시간(기본 30초)이 지나면 클립보드에서 자동으로 사라집니다. 다른 사람이 실수로 `Ctrl+V`를 눌러도 예전에 복사한 내 키 값이 나오지 않도록 하기 위한 안전장치입니다.

## 평소에 이렇게 사용하세요

- **프로그램을 켤 때**: 마스터 비밀번호를 입력해서 잠금을 해제합니다.
- **키가 필요할 때**: `Ctrl+K`로 검색 → 복사 아이콘 클릭 → 필요한 곳에 붙여넣기.
- **다 쓰고 나면**: `Ctrl+L`을 누르면 바로 잠깁니다. 안 눌러도 한동안 안 쓰면 자동으로 잠깁니다.

## 절대 하지 말아야 할 것

- **마스터 비밀번호와 복구키를 동시에 잊어버리지 마세요.** 이 프로그램은 개발자조차 대신 열어줄 수 없게 만들어졌습니다(그것이 보안입니다). 둘 다 잃어버리면 저장한 데이터를 되찾을 방법이 없습니다.
- **복구키를 이 프로그램 안에 다시 저장하지 마세요.** 복구키는 반드시 이 프로그램 "바깥"의 안전한 곳에 보관해야 의미가 있습니다.
- **낯선 사람에게 마스터 비밀번호나 복구키를 알려주지 마세요.** 이 둘을 아는 사람은 누구나 저장된 모든 키를 볼 수 있습니다.

## 문제가 생겼을 때

| 상황 | 이렇게 해보세요 |
|---|---|
| 설치할 때 파란 경고 화면이 떠요 | 정상입니다. "추가 정보" → "실행"을 순서대로 클릭하세요. ([2단계](#2단계--설치하기) 참고) |
| 비밀번호를 잊어버렸어요 | 로그인 화면에서 "복구키로 로그인"을 선택하고, 저장해뒀던 복구키를 입력하세요. |
| 복구키도 어디 있는지 모르겠어요 | 안타깝지만 저장된 데이터를 되찾을 방법이 없습니다. 프로그램을 다시 설치해서 새로 시작해야 합니다. |
| 프로그램이 실행되지 않아요 | 컴퓨터를 재시작해보세요. 그래도 안 되면 Windows 업데이트를 최신 상태로 만들어보세요. |

## 자주 묻는 질문

**Q. 인터넷이 없어도 쓸 수 있나요?**
A. 네. 다운로드할 때만 인터넷이 필요하고, 그 이후 사용할 때는 인터넷이 필요 없습니다.

**Q. 제 정보가 어딘가로 전송되나요?**
A. 아니요. 이 프로그램은 인터넷으로 아무것도 보내지 않도록 만들어졌습니다. 모든 정보는 내 컴퓨터 안에만 저장됩니다.

**Q. 스마트폰에서도 쓸 수 있나요?**
A. 아직은 컴퓨터(Windows)에서만 쓸 수 있습니다.

**Q. 다른 컴퓨터로 옮기고 싶으면 어떻게 하나요?**
A. 프로그램 안의 "설정 → 백업"으로 파일을 하나 만든 뒤, 그 파일을 옮긴 컴퓨터에서 "설정 → 복원"으로 불러오면 됩니다. 자세한 방법은 [README.md](./README.md#사용-방법--작동-방법)를 참고하세요.
