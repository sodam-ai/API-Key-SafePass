# API Key SafePass

> 在本地电脑上加密保管API密钥的桌面应用

无服务器、无云端 — 所有密钥仅保存在您的电脑上。

[English](./README.md) | [한국어](./README.ko.md) | [日本語](./README.ja.md)

---

## 这是什么应用？

使用ChatGPT、Claude、Gemini等AI服务时，API密钥散落在记事本、.env文件、浏览器标签等各处。**API Key SafePass**将所有密钥集中存放并加密保护。

**无需编程知识。** 安装后设置密码即可使用。

---

## 下载安装

1. 从[Releases](../../releases)页面下载安装程序
2. 双击 `API.Key.SafePass_x64-setup.exe`
3. 按向导点击"下一步"
4. 从开始菜单打开应用

> 应用大小：约2.5MB。无需网络连接。

## 主要功能

### 核心功能
- **AES-256-GCM加密** — 军事级加密
- **主密码** — 一个密码锁定所有密钥（Argon2id哈希）
- **恢复密钥** — 忘记密码时的应急密钥
- **项目管理** — 按项目分类管理

### 便捷功能
- **100+服务商预设** — OpenAI、Claude、Gemini、AWS、Stripe等一键选择
- **自动检测** — 粘贴密钥自动识别服务商
- **多语言搜索** — 中文、英文、韩文搜索
- **内联搜索** — 常驻搜索栏+结果计数
- **快速更新** — 无需完整表单，直接更换密钥值
- **重新签发** — 过期密钥一键跳转服务页面
- **参考URL** — 为每个密钥保存文档、定价链接
- **.env导出/导入**
- **自定义服务商** — 添加列表外的任何服务
- **无必填项** — 密钥值为空也可保存

### 安全性
- **仅限本地** — 无服务器、无云端、无网络传输
- **自动锁定** — 可配置（1/2/5/10/30分钟/关闭）
- **剪贴板自动清除** — 可配置（5/10/15/30/60秒）
- **屏幕模糊保护** — 切换窗口时模糊（可开关+延迟配置）
- **截图防护** — 操作系统级阻止
- **暴力破解防护** — 存储在数据库，重启也不重置
- **内存清零** — 锁定时加密密钥从内存完全删除
- **事务安全** — 密码更改崩溃也能保护数据

## 新手指南

### 第1步：安装
从[Releases](../../releases)下载安装程序并运行

### 第2步：设置主密码
1. 打开应用 → 创建密码（至少6个字符）
2. **恢复密钥**出现 → 务必保存到安全处
3. 点击"确认，已保存"

### 第3步：添加第一个密钥
1. 右下角 **+** 按钮 → 选择服务商 → 粘贴密钥 → 保存

### 第4步：日常使用

| 需求 | 方法 |
|------|------|
| 复制密钥 | "复制"按钮（自动清除剪贴板）|
| 搜索 | 顶部搜索栏 或 `Ctrl+K` |
| 锁定 | `Ctrl+L` 或侧边栏"锁定" |
| 更改密钥值 | 卡片上🔄按钮 |
| 设置 | 侧边栏 ⚙"设置" |

## 常见问题

**Q：忘记密码怎么办？**
使用设置时保存的恢复密钥。恢复密钥也丢失则无法恢复。

**Q：可以迁移到其他电脑吗？**
使用.env导出保存，在新电脑导入。

**Q：需要网络吗？**
不需要。100%离线运行。
