# API Key SafePass

> 在本地电脑上加密保管API密钥的桌面应用

无服务器、无云端 — 所有密钥仅保存在您的电脑上。

[English](./README.md) | [한국어](./README.ko.md) | [日本語](./README.ja.md)

---

## 这是什么应用？

使用ChatGPT、Claude、Gemini等AI服务时，API密钥散落在记事本、.env文件、浏览器标签等各处。**API Key SafePass**将所有密钥集中存放并加密保护。

**无需编程知识。** 安装后设置密码即可使用。

---

## 主要功能

### 核心功能
- **AES-256-GCM加密** — 军事级加密保护您的密钥
- **主密码** — 一个密码锁定所有密钥
- **恢复密钥** — 忘记密码时的应急密钥
- **项目管理** — 按"博客"、"电商"等项目分类管理

### 便捷功能
- **100+服务商预设** — OpenAI、Claude、Gemini、AWS、Stripe等一键选择
- **自动检测** — 粘贴密钥自动识别服务商
- **多语言搜索** — 支持中文、英文、韩文搜索
- **快速更新密钥** — 无需打开完整表单，直接更换密钥值
- **重新签发流程** — 过期密钥显示"重新签发"按钮，直接跳转服务页面
- **参考URL** — 为每个密钥保存文档、定价、控制台链接
- **.env导出** — 一键生成项目的.env文件
- **.env导入** — 从现有.env文件批量导入密钥

### 安全性
- **仅限本地** — 无服务器、无云端、无网络传输
- **自动锁定** — 可配置（1/2/5/10/30分钟）
- **剪贴板自动清除** — 可配置（5/10/15/30/60秒）
- **屏幕模糊保护** — 切换窗口时模糊处理（可配置）
- **截图防护** — 操作系统级屏幕捕获阻止
- **暴力破解防护** — 5次失败锁定60秒（存储在数据库中，重启应用也不会重置）
- **事务安全** — 密码更改过程中崩溃也能保护数据

## 新手指南

### 第1步：安装

1. 在项目文件夹中找到 `API Key SafePass_0.1.0_x64_en-US.msi`
2. 双击安装
3. 按照安装向导操作

### 第2步：初始设置

1. 从开始菜单或桌面打开 **API Key SafePass**
2. 创建**主密码**（至少6个字符）
3. **恢复密钥**出现时，务必复制到安全的地方
   - 此密钥不会再次显示！
4. 点击"确认，已保存"

### 第3步：添加第一个密钥

1. 点击右下角的大 **+** 按钮
2. 从下拉列表中**选择服务商**（如：OpenAI、Claude、Gemini）
   - 或者先粘贴密钥，服务商会被自动识别！
3. 名称、变量名、服务URL会自动填充
4. 点击**保存**

### 第4步：使用密钥

| 需求 | 方法 |
|------|------|
| 复制密钥 | 点击密钥卡片的"复制"按钮 → 剪贴板自动清除 |
| 搜索密钥 | 在顶部搜索栏输入 |
| 快速搜索 | `Ctrl+K` |
| 锁定应用 | `Ctrl+L` 或侧边栏"锁定" |
| 更改密钥值 | 鼠标悬停在卡片上 → 🔄按钮 |
| 更改设置 | 侧边栏底部 ⚙"设置" |
