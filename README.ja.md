# API Key SafePass

> APIキーを暗号化して安全に管理するデスクトップアプリ

サーバーなし、クラウドなし — すべてのキーはあなたのPCにのみ保存されます。

[English](./README.md) | [한국어](./README.ko.md) | [中文](./README.zh.md)

---

## このアプリは何ですか？

ChatGPT、Claude、Geminiなどを利用すると、APIキーがメモ帳、.envファイル、ブラウザのタブなどに散らばります。**API Key SafePass**はすべてのキーを一箇所に集めて暗号化します。

**プログラミング知識は不要です。** インストールしてパスワードを設定するだけ。

---

## ダウンロードとインストール

1. [Releases](../../releases)ページからインストーラーをダウンロード
2. `API.Key.SafePass_x64-setup.exe`をダブルクリック
3. ウィザードに従って「次へ」をクリック
4. スタートメニューからアプリを起動

> アプリサイズ：約2.5MB。インターネット接続不要。

## 主な機能

### コア機能
- **AES-256-GCM暗号化** — 軍事レベルの暗号化
- **マスターパスワード** — 一つのパスワードで全キーをロック（Argon2idハッシュ）
- **リカバリーキー** — パスワード紛失時の緊急キー
- **プロジェクト管理** — フォルダのように分類

### 便利な機能
- **100以上のプロバイダープリセット** — OpenAI、Claude、Gemini、AWS、Stripe他
- **自動検出** — キーを貼り付けるとプロバイダーを自動識別
- **多言語検索** — 日本語、英語、韓国語（初声）で検索可能
- **インライン検索** — 常に表示される検索バー＋結果カウント
- **クイックキー更新** — フォーム不要、キー値だけ変更
- **再発行フロー** — 期限切れキーに「再発行」ボタン
- **参考URL** — ドキュメント、料金、ダッシュボードリンク保存
- **.envエクスポート/インポート**
- **カスタムプロバイダー** — リストにないサービスも追加可能
- **必須フィールドなし** — キー値なしでも保存可能

### セキュリティ
- **ローカル専用** — サーバーなし、クラウドなし
- **自動ロック** — 設定可能（1/2/5/10/30分/オフ）
- **クリップボード自動削除** — 設定可能（5/10/15/30/60秒）
- **画面ぼかし保護** — ウィンドウ切り替え時にぼかし（ON/OFF＋遅延設定可能）
- **スクリーンショット防止** — OSレベルのキャプチャブロック
- **ブルートフォース防御** — DBに保存、アプリ再起動でもリセットされない
- **メモリゼロ化** — ロック時に暗号化キーをメモリから完全削除
- **トランザクション安全性** — パスワード変更中のクラッシュでもデータ保護

## 初めての方へ

### ステップ1：インストール
[Releases](../../releases)からインストーラーをダウンロードして実行

### ステップ2：マスターパスワード設定
1. アプリを起動 → パスワード作成（6文字以上）
2. **リカバリーキー**が表示 → 必ず安全な場所に保存
3. 「確認、保管しました」をクリック

### ステップ3：最初のキーを追加
1. 右下の **+** ボタン → プロバイダー選択 → キー貼り付け → 保存

### ステップ4：日常使用

| やりたいこと | 方法 |
|-------------|------|
| キーをコピー | 「コピー」ボタン（自動削除付き）|
| 検索 | 上部検索バー or `Ctrl+K` |
| ロック | `Ctrl+L` or サイドバー「ロック」|
| キー値変更 | カードの🔄ボタン |
| 設定 | サイドバー ⚙「設定」|

## よくある質問

**Q: パスワードを忘れた場合は？**
リカバリーキーで解除できます。リカバリーキーも紛失した場合は復旧できません。

**Q: 別のPCに移行できますか？**
.envエクスポートでキーをファイルに保存し、新PCでインポートしてください。

**Q: インターネットは必要ですか？**
不要です。100%オフラインで動作します。
