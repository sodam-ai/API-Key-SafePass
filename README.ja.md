# API Key SafePass

> APIキーを暗号化して安全に管理するデスクトップアプリ

サーバーなし、クラウドなし — すべてのキーはあなたのPCにのみ保存されます。

[English](./README.md) | [한국어](./README.ko.md) | [中文](./README.zh.md)

---

## このアプリは何ですか？

ChatGPT、Claude、Geminiなど多くのAIサービスを利用すると、APIキーがメモ帳、.envファイル、ブラウザのタブなど、あちこちに散らばってしまいます。**API Key SafePass**は、それらすべてのキーを一箇所に集めて暗号化して保管します。

**プログラミングの知識は不要です。** インストールしてパスワードを設定するだけで使えます。

---

## 主な機能

### コア機能
- **AES-256-GCM暗号化** — 軍事レベルの暗号化でキーを保護
- **マスターパスワード** — 一つのパスワードですべてのキーをロック
- **リカバリーキー** — パスワードを忘れた場合の緊急キー
- **プロジェクト管理** — 「ブログ」「ECサイト」のようにフォルダで分類

### 便利な機能
- **100以上のプロバイダープリセット** — OpenAI、Claude、Gemini、AWS、Stripeなどワンクリックで選択
- **自動検出** — キーを貼り付けるとプロバイダーを自動的に識別
- **多言語検索** — 日本語、英語、韓国語（初声）で検索可能
- **クイックキー更新** — 完全な編集フォームを開かずにキー値だけを変更
- **再発行フロー** — 期限切れのキーに「再発行」ボタン → サービスページへ直接移動
- **参考URL** — 各キーにドキュメント、料金ページ、ダッシュボードリンクを保存
- **.envエクスポート** — プロジェクトのキーを.envファイルとしてワンクリック生成
- **.envインポート** — 既存の.envファイルからキーを一括インポート

### セキュリティ
- **ローカル専用** — サーバーなし、クラウドなし、ネットワーク通信なし
- **自動ロック** — 設定可能（1/2/5/10/30分）
- **クリップボード自動削除** — 設定可能（5/10/15/30/60秒）
- **画面ぼかし保護** — ウィンドウを切り替えると画面をぼかし処理（設定可能）
- **スクリーンショット防止** — OS レベルの画面キャプチャブロック
- **ブルートフォース防御** — 5回失敗で60秒ロック（DBに保存、アプリ再起動でもリセットされない）
- **トランザクション安全性** — パスワード変更中のクラッシュでもデータを保護

## 初めての方へ

### ステップ1：インストール

1. プロジェクトフォルダから `API Key SafePass_0.1.0_x64_en-US.msi` を見つけてください
2. ダブルクリックでインストール
3. インストールウィザードに従ってください

### ステップ2：初期設定

1. スタートメニューまたはデスクトップから **API Key SafePass** を起動
2. **マスターパスワード**を作成（6文字以上）
3. **リカバリーキー**が表示されたら、必ず安全な場所にコピーしてください
   - このキーは二度と表示されません！
4. 「確認、保管しました」ボタンを押す

### ステップ3：最初のキーを追加

1. 画面右下の大きな **+** ボタンを押す
2. ドロップダウンから **プロバイダーを選択**（例：OpenAI、Claude、Gemini）
   - またはキーを先に貼り付ければ自動検出されます！
3. 名前、変数名、サービスURLが自動入力されます
4. **保存**を押す

### ステップ4：キーを使う

| やりたいこと | 方法 |
|-------------|------|
| キーをコピー | キーカードの「コピー」ボタン → クリップボードから自動削除 |
| キーを検索 | 上部の検索バーに入力 |
| クイック検索 | `Ctrl+K` |
| アプリをロック | `Ctrl+L` またはサイドバーの「ロック」 |
| キー値を変更 | カードにマウスオーバー → 🔄ボタン |
| 設定変更 | サイドバー下部の ⚙「設定」 |
