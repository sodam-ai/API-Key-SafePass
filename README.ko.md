# API Key SafePass

> 내 PC에서 API 키를 암호화하여 안전하게 보관하는 데스크톱 앱

서버 없음, 클라우드 없음 — 모든 키는 내 컴퓨터에만 저장됩니다.

[English](./README.md) | [日本語](./README.ja.md) | [中文](./README.zh.md)

---

## 이 앱은 뭐예요?

ChatGPT, Claude, Gemini 같은 AI 서비스를 쓰다 보면 API 키가 메모장, .env 파일, 브라우저 탭 여기저기에 흩어져 있습니다. **API Key SafePass**는 이 키들을 한 곳에 모아 암호화해서 보관합니다.

**코딩을 몰라도 됩니다.** 설치하고 비밀번호 설정하면 바로 쓸 수 있습니다.

---

## 주요 기능

### 핵심
- **AES-256-GCM 암호화** — 군사급 암호화로 키를 보호
- **마스터 비밀번호** — 하나의 비밀번호로 모든 키를 잠금
- **복구키** — 비밀번호를 잊었을 때 사용하는 비상키
- **프로젝트별 정리** — "내 블로그", "쇼핑몰" 처럼 폴더로 분류

### 편의 기능
- **100개+ 제공자 프리셋** — OpenAI, Claude, Gemini, AWS, Stripe 등 클릭 한 번으로 선택
- **자동 감지** — 키를 붙여넣으면 어떤 서비스의 키인지 자동으로 알아냄
- **한국어/초성 검색** — "오픈" 또는 "ㅇㅍ"으로 OpenAI 검색 가능
- **키값 빠른 변경** — 전체 수정 없이 키값만 바로 바꾸기
- **재발급 흐름** — 만료된 키에 "재발급" 버튼 → 서비스 페이지로 바로 이동
- **참고 URL** — 각 키에 문서, 요금 페이지, 대시보드 링크 저장
- **.env 내보내기** — 프로젝트의 키를 .env 파일로 한 번에 생성
- **.env 가져오기** — 기존 .env 파일에서 키를 한 번에 가져오기

### 보안
- **로컬 전용** — 서버 없음, 클라우드 없음, 네트워크 전송 없음
- **자동 잠금** — 설정 가능 (1분/2분/5분/10분/30분)
- **클립보드 자동 삭제** — 설정 가능 (5초/10초/15초/30초/60초)
- **화면 흐림 보호** — 다른 창으로 전환하면 화면 흐리게 처리 (설정 가능)
- **스크린샷 방지** — OS 수준 화면 캡처 차단
- **무차별 대입 방어** — 5회 실패 시 60초 잠금 (DB에 저장, 앱 재시작해도 유지)
- **복구키 시도 제한** — 3회 실패 시 5분 잠금
- **메모리 제로화** — 잠금 시 암호화 키를 메모리에서 완전 삭제
- **트랜잭션 안전** — 비밀번호 변경 중 크래시해도 데이터 보호

## 처음 사용하는 분을 위한 안내

### 1단계: 설치

1. 프로젝트 폴더에서 `API Key SafePass_0.1.0_x64_en-US.msi` 파일을 찾으세요
2. 더블클릭해서 설치하세요
3. 설치 마법사를 따라가세요

### 2단계: 초기 설정

1. 시작 메뉴 또는 바탕화면에서 **API Key SafePass**를 실행하세요
2. **마스터 비밀번호**를 만드세요 (6자 이상)
3. **복구키**가 나타나면 반드시 메모장이나 안전한 곳에 복사해두세요
   - 이 복구키는 다시 볼 수 없습니다!
   - 비밀번호를 잊었을 때 유일한 복구 수단입니다
4. "확인, 보관했습니다" 버튼을 누르세요

### 3단계: 첫 번째 키 추가

1. 화면 오른쪽 아래의 큰 **+** 버튼을 누르세요
2. **제공자를 선택**하세요 (예: OpenAI, Claude, Gemini)
   - 드롭다운에서 한국어로 검색할 수 있어요 ("오픈에이아이", "클로드")
   - 또는 키를 먼저 붙여넣으면 자동으로 제공자가 감지됩니다!
3. 이름, 변수명, 서비스 URL이 자동으로 채워집니다
4. **저장** 버튼을 누르세요
   - 이름이 비어있어도 자동으로 만들어집니다
   - 키값이 비어있어도 저장됩니다 (나중에 추가 가능)

### 4단계: 키 사용하기

| 하고 싶은 것 | 방법 |
|-------------|------|
| 키 복사 | 키 카드의 "복사" 버튼 클릭 → 자동으로 클립보드에서 삭제됨 |
| 키 검색 | 상단 검색바에 입력 (한글, 영문, 초성 모두 됨) |
| 빠른 검색 | `Ctrl+K` 누르기 |
| 앱 잠금 | `Ctrl+L` 또는 사이드바 "잠금" 클릭 |
| 키값 변경 | 키 카드에 마우스 올리기 → 🔄 버튼 → 새 키값 입력 |
| 만료된 키 재발급 | 빨간 "재발급" 버튼 → 서비스 페이지 열기 → 새 키 입력 |
| 설정 변경 | 사이드바 하단 ⚙ "설정" 클릭 |
| 비밀번호 변경 | 설정 → 마스터 비밀번호 변경 |

## 자주 묻는 질문

### Q: 비밀번호를 잊어버리면?
복구키를 입력하면 잠금을 해제할 수 있습니다. 복구키도 잃어버렸다면 안타깝지만 복구할 수 없습니다 (이것이 보안의 핵심입니다).

### Q: 다른 PC로 옮길 수 있나요?
현재는 .env 내보내기/가져오기로 키를 이동할 수 있습니다. 암호화 백업 기능은 향후 추가 예정입니다.

### Q: 인터넷이 필요한가요?
아니요. 이 앱은 100% 오프라인으로 작동합니다. 키가 네트워크를 통해 전송되는 일은 없습니다.

### Q: 앱이 안 열려요
마스터 비밀번호를 5번 틀리면 60초간 잠깁니다. 잠시 기다렸다가 다시 시도하세요.
