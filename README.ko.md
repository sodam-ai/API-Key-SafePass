# API Key SafePass

> 내 PC에서 API 키를 암호화하여 안전하게 보관하는 데스크톱 앱

서버 없음, 클라우드 없음 — 모든 키는 내 컴퓨터에만 저장됩니다.

[English](./README.md) | [日本語](./README.ja.md) | [中文](./README.zh.md)

---

## 이 앱은 뭐예요?

ChatGPT, Claude, Gemini 같은 AI 서비스를 쓰다 보면 API 키가 메모장, .env 파일, 브라우저 탭 여기저기에 흩어져 있습니다. **API Key SafePass**는 이 키들을 한 곳에 모아 암호화해서 보관합니다.

**코딩을 몰라도 됩니다.** 설치하고 비밀번호 설정하면 바로 쓸 수 있습니다.

---

## 다운로드 및 설치

1. [Releases](../../releases) 페이지에서 설치파일 다운로드
2. `API.Key.SafePass_x64-setup.exe` 또는 `.msi` 실행
3. 설치 마법사 따라가기 ("다음" 클릭)
4. 시작 메뉴에서 **API Key SafePass** 실행

> 앱 크기: ~2.5 MB. 인터넷 연결 불필요.

## 주요 기능

### 핵심
- **AES-256-GCM 암호화** — 군사급 암호화로 키를 보호
- **마스터 비밀번호** — 하나의 비밀번호로 모든 키를 잠금 (Argon2id 해싱)
- **복구키** — 비밀번호를 잊었을 때 사용하는 비상키
- **프로젝트별 정리** — "내 블로그", "쇼핑몰" 처럼 폴더로 분류

### 편의 기능
- **100개+ 제공자 프리셋** — OpenAI, Claude, Gemini, AWS, Stripe 등 클릭 한 번으로 선택
- **자동 감지** — 키를 붙여넣으면 어떤 서비스의 키인지 자동으로 인식
- **한국어/초성 검색** — "오픈" 또는 "ㅇㅍ"으로 OpenAI 검색 가능
- **인라인 검색** — 항상 보이는 검색바 + 결과 카운트
- **키값 빠른 변경** — 전체 수정 없이 키값만 바로 바꾸기
- **재발급 흐름** — 만료된 키에 "재발급" 버튼 → 서비스 페이지로 바로 이동
- **참고 URL** — 각 키에 문서, 요금 페이지, 대시보드 링크 저장
- **.env 내보내기/가져오기** — 프로젝트의 키를 .env 파일로 생성하거나 기존 .env에서 가져오기
- **직접 추가** — 목록에 없는 제공자도 자유롭게 등록
- **빈칸 허용** — 키값 없이도 저장 가능 (나중에 추가)

### 보안
- **로컬 전용** — 서버 없음, 클라우드 없음, 네트워크 전송 없음
- **자동 잠금** — 설정 가능 (1분/2분/5분/10분/30분/사용 안 함)
- **클립보드 자동 삭제** — 설정 가능 (5초/10초/15초/30초/60초)
- **화면 흐림 보호** — 다른 창으로 전환 시 화면 흐림 (ON/OFF + 딜레이 설정 가능)
- **스크린샷 방지** — OS 수준 화면 캡처 차단
- **무차별 대입 방어** — DB에 저장되어 앱 재시작해도 유지
  - 비밀번호: 5회 실패 → 60초 잠금
  - 복구키: 3회 실패 → 5분 잠금
- **메모리 제로화** — 잠금 시 암호화 키를 메모리에서 완전 삭제
- **URL 스킴 검증** — http/https만 허용 (javascript: 주입 차단)
- **트랜잭션 안전** — 비밀번호 변경 중 크래시해도 데이터 보호
- **설정 키 허용 목록** — 백엔드에서 4개 설정만 수정 가능
- **에러 메시지 일반화** — 내부 정보 노출 차단

### 설정 (사용자 조절 가능)
- 자동 잠금 시간
- 클립보드 삭제 시간
- 화면 흐림 ON/OFF + 딜레이 (즉시/3초/5초/10초)
- 마스터 비밀번호 변경 (강도 표시기 포함)

## 처음 사용하는 분을 위한 안내

> **코딩 경험이 전혀 없어도** 따라할 수 있습니다.

### 1단계: 설치

1. [Releases](../../releases) 페이지에서 설치파일 다운로드
2. `API.Key.SafePass_x64-setup.exe` 더블클릭
3. 설치 마법사에서 "다음"만 누르면 됩니다
4. 시작 메뉴에 앱이 나타납니다

### 2단계: 마스터 비밀번호 설정

1. **API Key SafePass** 실행
2. 잠금 화면이 나옵니다 — 처음이니까 비밀번호를 만드세요
3. 비밀번호 입력 (6자 이상) + 확인
4. **중요!** 화면에 **복구키**가 나타납니다
   - 메모장, 비밀번호 관리자, 종이 등 안전한 곳에 복사해두세요
   - **이 키는 다시 볼 수 없습니다!**
   - 비밀번호를 잊었을 때 유일한 복구 수단입니다
5. "확인, 보관했습니다" 클릭

### 3단계: 첫 번째 키 추가

**방법 1: 제공자 먼저 선택**
1. 오른쪽 아래 큰 **+** 버튼 클릭
2. "제공자 선택" 드롭다운 클릭
3. 서비스 검색 (예: "OpenAI" 또는 "오픈" 또는 "ㅇㅍ")
4. 클릭하면 이름, URL, 변수명이 자동으로 채워짐
5. 키값 붙여넣기
6. **저장** 클릭

**방법 2: 키 먼저 붙여넣기 (자동 감지)**
1. **+** 버튼 클릭
2. "키값" 필드에 키 붙여넣기
3. `sk-`로 시작하면 OpenAI, `sk-ant-`이면 Claude 등 자동 감지
4. **저장** 클릭

**방법 3: 나중에 키 추가**
1. **+** 버튼 → 제공자 선택 → **저장** (키값 비워도 됨)
2. 나중에 키 카드의 🔄 버튼으로 키값만 추가

### 4단계: 일상적인 사용

| 하고 싶은 것 | 방법 |
|-------------|------|
| **키 복사** | 키 카드의 "복사" 버튼 (자동으로 클립보드에서 삭제됨) |
| **키 검색** | 상단 검색바에 입력 (한글, 영문, 초성 모두 됨) |
| **빠른 검색+복사** | `Ctrl+K` → 검색 → `Enter`로 바로 복사 |
| **앱 잠금** | `Ctrl+L` 또는 사이드바 "잠금" 클릭 |
| **키값 변경** | 키 카드에 마우스 올리기 → 🔄 버튼 → 새 키값 입력 |
| **만료된 키 재발급** | 빨간 "재발급" 버튼 → 서비스 페이지 열기 → 새 키 입력 |
| **.env 내보내기** | 프로젝트 선택 → 헤더의 ".env" 버튼 |
| **설정 변경** | 사이드바 ⚙ "설정" → 잠금/흐림/클립보드 조절 |
| **비밀번호 변경** | 설정 → "마스터 비밀번호 변경" |

## 자주 묻는 질문

**Q: 비밀번호를 잊어버리면?**
설정할 때 받은 복구키를 입력하면 잠금을 해제할 수 있습니다. 복구키도 잃어버렸다면 복구할 수 없습니다 (보안을 위한 설계입니다).

**Q: 비밀번호 변경 후 복구키가 안 됩니다**
비밀번호를 변경하면 기존 복구키는 무효화됩니다. 새 복구키를 설정해야 합니다.

**Q: 다른 PC로 옮길 수 있나요?**
.env 내보내기로 키를 파일로 저장한 뒤, 새 PC에서 .env 가져오기를 하면 됩니다.

**Q: 인터넷이 필요한가요?**
아니요. 100% 오프라인 앱입니다. 키가 네트워크를 통해 전송되는 일은 없습니다.

**Q: 앱이 잠겨서 안 열려요**
비밀번호를 5번 틀리면 60초간 잠깁니다. 잠시 기다렸다가 다시 시도하세요.
